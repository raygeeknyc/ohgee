Screen connections
   * VCC : 3.3V
   * Gnd : ground
   * CS : CEO
   * Reset : GOIO25
   * D/C : GPIO24
   * SDI/MOSI : MOSI
   * SCK : SCLK
   * LED : GPIO23 # default was 18 but that PWM is needed for the Servo
   
Servo connections
   * Gnd : ground
   * VCC : 5V
   * Signal : GPIO18
   
LED connections
   * Anode : 3.3V
   * Red cathode : GPIO2
   * Green cathode : GPIO3
   * Blue cathode : GPIO4
